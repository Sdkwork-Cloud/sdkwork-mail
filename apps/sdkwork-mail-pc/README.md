# SDKWork Mail PC Application

PC browser/desktop application for SDKWork Mail. Supports mailbox browsing, compose/send, folders, threads, verification mail, and Drive-backed attachments.

## Architecture

- Root: `APP_PC_ARCHITECTURE_SPEC.md`
- SDK: `sdkwork-mail-app-sdk` (generated)
- Packages: `sdkwork-mail-pc-core`, `sdkwork-mail-pc-commons`, `sdkwork-mail-pc-shell`, `sdkwork-mail-pc-mail`

## Development

```powershell
pnpm install
pnpm run dev
```
