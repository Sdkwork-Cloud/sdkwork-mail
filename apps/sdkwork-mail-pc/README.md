# SDKWork Mail PC Application

PC browser/desktop application for SDKWork Mail. Supports rooms, mail inboxs, participant credentials, provider profiles, and recording artifacts.

## Architecture

- Root: `APP_PC_ARCHITECTURE_SPEC.md`
- SDK: `sdkwork-mail-app-sdk` (generated)
- Packages: `sdkwork-mail-pc-core`, `sdkwork-mail-pc-commons`, `sdkwork-mail-pc-shell`, `sdkwork-mail-pc-Mail`

## Development

```powershell
pnpm install
pnpm run dev
```
