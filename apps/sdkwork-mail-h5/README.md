# SDKWork Mail H5 Application

H5 mobile web application for SDKWork Mail. Phone-first touch interface for rooms, mail inboxs, and participant management.

## Architecture

- Root: `APP_H5_ARCHITECTURE_SPEC.md`
- SDK: `sdkwork-mail-app-sdk` (generated)
- Packages: `sdkwork-mail-h5-core`, `sdkwork-mail-h5-commons`, `sdkwork-mail-h5-shell`, `sdkwork-mail-h5-Mail`

## Development

```powershell
pnpm install
pnpm run dev
```
