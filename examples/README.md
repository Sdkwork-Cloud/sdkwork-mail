# Mail Examples

## Purpose

`examples/` is reserved for maintained Mail API, SDK, plugin, and runtime usage examples.

## Owner

sdkwork-mail.

## Allowed Content

- Runnable examples.
- SDK usage examples.
- Safe sample configs and fixtures.

## Forbidden Content

- Generated SDK transport output.
- Production secrets, account identifiers, or private provider credentials.
- Unmaintained scratch prototypes.
